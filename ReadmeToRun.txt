Readme

To run the agent, run the command 
`java -jar BamBird.jar $hostname$ $teamid$ $pathtoprolog$`

$hostname$ - name of the server;
$teamid$ - ID of the BamBirds team;
$pathtoprolog$ - path to SWI-Prolog (swipl) in your machine;

Important: 
----------

The directory "Prolog" has to be in the same directory as the 
.jar-file! 

Also, the path to the "Prolog"-directory must not contain a space character!
