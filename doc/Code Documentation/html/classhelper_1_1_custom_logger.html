Detailed Description

This is a custom-made logger, it uses java.utils.logging. It can be used in any class via static call "CustomLogger.info(msg)". The calling class will automatically be derived by the log methods (the line with "exception"). Call CustomLogger.info(), ~.warning() or ~.severe() for different logging levels. Default log level is "info", but this can be changed in any class via "setLogLevel(String level)" with "info", "warning" or "severe". (recommended is of course the main entry point for setting the log level).
