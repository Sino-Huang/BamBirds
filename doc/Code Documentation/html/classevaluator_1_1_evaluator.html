Detailed Description

    This class is used for evaluating the performance of AI-Bird agents. It is assumed that there is a pre-existing file (logs/logs.csv),
    which can be generated with main.py.
    If you want to choose by yourself what agents and levels should be evaluated, run python main.py.
    If you want to evaluate with the default settings, run python main.py d
    This class generates bar charts which contain statistics about the performance of the agents. These charts can be generated in a pdf and as single images.
    The class also outputs the statistics as a csv file

private 
This method evaluates the performance of the given agents on the given level. It generates output as a pdf file and optionally as single svg files.
Before this method is run, there has to be an open PdfPages document. Afterwards, when all levels are evaluated, the document has to be closed.
:param level: the id of the level to be evaluated.
:param agents: The Ai Birds agents to be evaluated. Only those agents are available that are mentioned in the log csv file
:param level_set: The group of levels to be evaluated, i.e the name of the directory where the levels are located (e.g., quarter-final_competition_2017), only those level sets are available that are mentioned in the log csv file
:param save_single_files: If True, the generated bar charts are not only found in a pdf file, but also as single svg images in the plots directory
