List<Target> planner.NewPrologPlanParser.parsePlans ( String  plans)

Parses a Prolog plan of the new (2017) format. The Prolog-module of our agent only returns a String representation of his results, which has to be processed into Targets.

The format used by Prolog is: [[target,chosenStrat,confidence],...,[targetN,chosenStratN,confidenceN]]
